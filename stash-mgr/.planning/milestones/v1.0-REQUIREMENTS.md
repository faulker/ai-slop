**Archived:** 2026-02-12

## v1 Requirements (All Shipped)

### Foundation

- [x] **FNDN-01**: Application detects and opens the git repository from the current directory
- [x] **FNDN-02**: Application sets up terminal (raw mode, alternate screen) and restores on exit/panic
- [x] **FNDN-03**: User can switch between Create Stash and Manage Stashes tabs using Tab key
- [x] **FNDN-04**: User can quit the application with 'q' key

### Stash Browsing

- [x] **BRWS-01**: User can see a list of all stashes with index, message, and branch name
- [x] **BRWS-02**: User can navigate the stash list with arrow keys
- [x] **BRWS-03**: User can see a diff preview of the selected stash in a side panel
- [x] **BRWS-04**: User can apply a selected stash (keep stash in list)
- [x] **BRWS-05**: User can pop a selected stash (apply and remove from list)
- [x] **BRWS-06**: User can drop (delete) a selected stash with confirmation

### Stash Creation

- [x] **CREA-01**: User can see a list of modified and staged tracked files (no untracked)
- [x] **CREA-02**: User can navigate the file list with arrow keys
- [x] **CREA-03**: User can toggle file selection with checkboxes (space key)
- [x] **CREA-04**: User is prompted for a stash message before creating the stash
- [x] **CREA-05**: User can create a stash from selected files with the entered message
- [x] **CREA-06**: File list refreshes after a stash is created
