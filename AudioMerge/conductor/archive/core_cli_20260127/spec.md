# Specification: Core CLI & Scanning

## Goal
Establish the foundational CLI structure and the recursive directory traversal logic.

## Requirements
- Scaffold Rust project with `cargo`.
- Define CLI arguments using `clap` (source, output, dry-run, on-error).
- Implement recursive file system walker.
- Implement intelligent sorting logic for filenames.
- Implement dry-run visualization (Table/List format).
